# ts-starter-template

## Usage

Template repo with base TS, ESLint and PNPM

### Prerequisites

- IDE VSCode
  - Prettier
- Node
- PNPM
- git
- Optional
  - Github CLI

### Installation

1. With GitHub CLI:

```bash
gh repo create <project-name> --clone <--public | --private | --internal> --template ts-starter-template

cd <project-name>
pnpm i
```

2. Update package.json
